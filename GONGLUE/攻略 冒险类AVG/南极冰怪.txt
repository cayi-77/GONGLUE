序言～北极星计划 

　　皇家海军维多利亚号潜艇，载著二个神秘木箱及一位娜威探险家毕尔汉森；但遗憾的，北极星计划突击小组却无法救出汉森的父亲－－彼得他将永远被囚禁在那绝对机密的基地中 

１９３７年１月－－－南极海底～维多利亚号潜艇 

　　北极星计划虽己完成，但毕尔汉森精神状况非常危急；欧利莱上尉副舰长失踪，雷恩上尉（此游戏主角）也就顺位接替了欧利莱副舰长的职位

　　向左边的洛埃舰长问有关汉森、木箱、北极星计划的事，但舰长却不怎麽乐意回答；此时声纳雷达侦测到有敌舰接近，舰长马上下令潜航，虽避开了部分的深水炸弹，但舰尾仍被击中了，舰长便要我和其一起至货舱查看受损情形。 

　　一到货舱便见火海一片；上前查看躺在地上生死未卜的锺斯，但锺斯只说了『在木箱里面有东西』便拜拜了舰长看见有一道密室门打开，便上前查看，而覆盖在木箱上的冰因大火的关系渐渐溶化。 

　　此时，从密室门内突然伸出一只怪手，把洛埃舰长卷进去，一看不妙马上把密室门关上，顺手拿了上方墙上的灭火器把大火扑灭後，便马上回到舰桥告诉德斯可有关锺斯及舰长的死讯及木箱出现怪物。 

──────　 │２－冰怪初现│ 　──────　 

　　在德斯可告知下得知敌舰己消失，目前最大问题便是想办法对付木箱的怪物，但德斯可却说舰上并没有攻击性的武器，并给了一支无线电手机以便联络走到舰桥左上方的书桌抽屉找到一台录音机、密码簿、及一把钥匙；从右侧门到卧舰看见韦恩正在看顾汉森；四处找找後，在右下方桌子找到一条圣克里斯多夫奖章，左下卧下找到雪鞋、救生衣，左上方墙上拿到手斧走到汉森卧，便叫韦恩到舰桥看德斯可有什麽要帮忙的；此时汉森在那边喃喃自语，不知说啥？便拿著圣克里斯多夫奖章对汉森催眠，之後汉森便念了一句像咒语的语言，赶紧拿录音机将此咒语录下回到舰桥，忽然一声巨响，左侧门飞了出来，正好击中韦恩（挂了！衰！），而从左侧门也出现一只怪物，正在不知所措之时，拿出录音机（使用在自己身上）播放出汉森所念的咒语，可真玄！！！怪物竟被消灭了此时舰首传来爆炸声，但并不见敌舰及暗礁的踪迹，便怀疑是自己人所破坏；问德斯可如何发出求救信号，他说在引擎室的史坦利可能知道无线电频率。 

────　 │３－求救│ 　────　 

　　从右下门下到引擎室，却见到史坦利被钢梁压住，并得知须使用绞盘方能吊起钢梁；打开右边的开关，再使用无线电手机通知德斯可启动绞盘吊起钢梁，之後向史坦利问得无线电频率临走时顺手捡走地上的活动扳手（在中间机器的下方处）回到舰桥，使用活动板手拆了地上舱门上的金属轮；再使用密码簿在左方桌子上的无线电发出求救信号；虽然友舰收到求救信号，但因舰内进水无法上浮，必须想办法到水面上送出信号好让友舰发现所在；在与德斯可商谈後，决定利用鱼雷室的鱼雷浮出水面先左走到货舱，发现地面全是冰无法通过，便穿上雪鞋走到左方，用钥匙打开墙上架子上的锁拿到信号枪走回到卧舱，想打开门到鱼雷室，但因电路受损，门上的电子锁己自动锁上了；回到舰桥，用手斧打开右墙上的电气箱，将电路接通（如图）即可打开鱼雷室的门。

　　１９３７年１月１３日－－－福克兰群岛～爱德华基地在基地与西尔斯舰长谈完话後，便在办公室等候补给官昆西；四处找找，在右上方书桌抽屉找到一张只有一半写著『５２３』的密码纸，桌上拿到自己的人事卡片及一包香烟，此时昆西也进来，交代了勤务簿後，我便走出办公室了。

　　出了办公室後，把勤务簿交给门口守卫；此时警报响起，要所有守卫前往地下室；没人了，便把放在守卫桌上由潜伏在德国的ＳＰＹ莫莉小姐寄来的影片拿走进右上门到通信室，电信员萧不理你；看见後面正在烧开水，无聊便把自己的人事卡片放在上面蒸热气，相片就拿下来了出通信室，进右门到大厅，再进右下门到简报室走廊，再由另一门进到简报室看见麦克连；一问之下，才知其被昆西禁烟；拿香烟给他，再拿影片给他播放，则怪物之谜就可得知了顺便在上方书架的书中找到另一半密码『４９６』（在左上那格，左边第５本） 

　　回到大厅，遇见特雷弗医生说汉森失踪，并在其床尾发现绿色黏液说完便坐电梯走了回电信室找萧，拿到一份来自华盛顿的电报『在基地内有间谍，去查查基地的个人档案，并注意汉森』；问萧档案室在那里，他说在军械室隔壁。 

　　回到大厅坐电梯到地下室，但地下室守卫说要有通行证才能通过；回到西尔斯的办公室，打开左方墙上的图画发现保险箱，使用找到的密码『４９６５２３』便可打开，拿走一把洛埃的钥匙及像皮图章；再把相片贴在桌上的通行证并盖上像皮图章即成一张正式的通行证；把保险箱恢复原状，再到地下室去。 

───────────　│２－爱德华基地地下室 │　───────────　 

　　地下室守卫看了通行证便让你通过进到守卫室，再进右上门到储藏室，在上方架子上拿到一个锡罐及打开旁边的大箱子，看见一具满身绿色黏液的死尸回到守卫室，再进右下门到等候室，假装生病跟特莲小姐预约挂号见医生；见到医生後骗其说胃不舒服，再把锡罐拿给他看，医生就会进去配药，趁此机会赶快拿走右方桌上的组合手册，拿药後便可离开了。 

　　回到守卫室，敲敲上方的门，门上便开一小窗，把组合手册给芬雷森，他就会开门让你进去了；进到军械室问芬雷森，他说要舰长才能进档案室；灵机一动，拿走左上墙上的灭火器，再拿桌上点燃的香烟丢进右方垃圾桶里，再躲在右方黑暗角落处；结果著火了，芬雷森找不到灭火器，便叫档案室内的玛西一起逃命；等他们出去後，先用灭火器把火扑灭再进档案室；打开档案抽屉才知有人捷足先登拿走所有档案，只剩下一个名叫帕克的档案文件。 

　　要离开军械室时遇见西尔斯，夸奖我消灭火源并请我到他的办公室；相谈之下，得知基地出现怪物，而西尔斯得知咒语能对付怪物，但录音机在离开潜艇时己经遗失了，所以便叫我回维多利亚潜艇找找有没有汉森留下的东西。 

────────── │３－再访维多利亚潜艇│　──────────　 

　　回到舰艇上想要进舰桥，但己经锁上了；走到右边捡起地上的电线，将其绑在右边棒子上，则可走到舰艇最右方，打开地板上的柜子拿到二支金属工具，将此二支金属工具组合，则变成艇盖钥匙，去开舰艇入口的门。 

　　进入到卧舱，在左上方橱柜找到一份有关咒文的汉森笔记纸；回到舰桥，怪物突然出现，此时赶快跑到驾驶台（原先德斯可操作的地方）左方有个按钮，把洛埃的钥匙插进去打开开关，则警报响起，趁怪物分心之时爬上梯子离开潜舰。 

──────────　│４－冰怪！！汉森！！│ 　──────────　 

　　回到基地，即见守卫来说电信员萧被杀，西尔斯失踪，这时特雷弗医生出现帮我解围，并请我到医务所看汉森的血液化验，得知汉森血液非比寻常，并且看见汉森的小手册，得知只要在地上画五星芒记号即可阻止怪物，而有一颗叫Ｍａｎｒｓｔｏｎｅ的石头可将怪物送回太虚；此时护士从医务室出来，一脸惊慌像；赶紧进医务室，看见昆西爬在地上说了『石头在地图後面』後，即被怪物吃下肚去了；赶紧在地上画上五星芒记号阻止怪物行动；再回到手术室桌上拿到一根针（原先拿组合手册处），再到西尔斯办公室查看上方的南极洲大地图，发现中央处有个红色小洞，将针插入即启动机关，拿到一块石头及一份秘密文件；回医务室使用石头（用在自己身上）则怪物就消失在一番对话後，得知西尔斯是叛徒，且接获一封华盛顿电报『到布宜诺斯艾利斯图书馆找一份ＯＴＲ　２８３２的重要参考考文件，并要尽快找到帕克』。

１９３７年１月１６日－－－　布宜诺斯艾利斯～图书馆 　 

　　进到图书馆後，问管理员赫尼德兹，得知太阳石盘被偷走；再跟他说要寻找ＯＴＲ　２８３２文件，便可进去找馆长佐治，而在门口遇见一位小姐，一问才知其为帕克儿黛安，便和我一起，寻找失踪的帕克。 

───────　│１－另一个自己│─────── 

　　进去见到馆长，得知ＯＴＲ２８３２是件古老研究文件，且现在所见的乃是唯一的复制版本，并知帕克被怀疑偷走太阳石盘而被列为嫌疑犯，但佐治馆长不相信此说，他认为帕克可能是因为某种原因而将太阳石盘藏在某处； 

　　接著他便叫米吉尔叙述一件古老的传奇正讲到一半时，一个名叫哈伦德的人拿著枪冲了进来，说要太阳石盘，管理员赫尼德兹进来劝阻之际，便遭一枪毙命！突然，另一个『我』出现，拿枪把哈伦德干掉了，并说了一些话便消失了； 

　　此时馆长说警察马上来到，便叫我拿走ＯＴＲ２８２３文件与黛安躲进密门内；警察来到找不到我们便把馆长带走了。 

─────────　│２－图书馆储藏室　│ 　─────────　 

　　Ⅰ：第一层从密门出来後，黛安说要到顶坛找线索，随著黛安走到一间图书储藏室；在左边墙下拿到一只手杖及一堆书（哥德的书，沙孚克里斯的书，莎士比亚的书），在书架上找到一本书（在人头石像下方书柜的的第三横格约中间处） 

　　打开机关出现通道，来到第二层Ⅱ：第二层在此层有一梯子，但其中一横木坏了，便使用手杖代替。 

　　上到第三层Ⅲ：第三层，在此层中间书柜有一空的空间，将拿到的书，由左至右依序摆上沙孚克里斯的书、莎士比亚的书、哥德的书，即出现一阶梯。 

　　上到第四层Ⅳ：第四层，此层直接往右走即可找到梯子。 

　　上到第五层Ⅴ：第五层，此层有五座雕像，查最右边的雕像，则中间雕像打开一暗格，拿到一把钥匙，开门到顶坛Ⅵ：顶坛在顶坛有三尊雕像，先爬上左边雕像，跳到中间雕像，拿右边雕像手上的太阳石盘；此时几位纳走狗拿枪来将我们二人押回其基地监狱中。 

１９３７年１月１７日－－－舒罗沙勒基地～监狱 

　　在监狱里，隔壁传来要我挖墙说有重要事要告知，拿起桌上的汤匙挖墙（要避开警卫的查房），才知隔壁房是帕克，并知纳准备用邪恶力量统治全世界；之後见到他们一个个被带走，此时纳军官戴奇进来给了一份假文件及一支钢笔要我十分钟内考虑决定；反正签不签都是死路一条！便把假文件塞在洗手盆并打开水，让水流出外面，警卫一看便进来查看，顺手拿起右边的椅子把警卫打昏（站的地方离门远点），在他身上找到钥匙，赶快把门关上，再把桌子推到中间，将椅子放在上面，爬上去用钢笔打开通风气孔，爬进通风口滑落到四处都是岩浆的地方。 

──────　 │１－溶岩地带│ 　──────　 

　　去移动上方的石笋，则左方石门打开，拿到红宝石、紫水晶；再往右走到一人头石像前，将红宝石、紫水晶放进其眼睛中，则石像嘴巴打开，进入被传送到另一地方往右走看见一台矿车，在车内找到一根矿工的棒子，而矿车车轮被冰块冻住无法推动；往左走到那被一堆石块堵住溶浆的下方，发现一块松动的石头，用矿工的棒子将其撬开，则岩浆便流下来；将棒子放进加热，再用棒子把车轮的冰块溶化，再将矿车推动，则铁门也打开了；此时有块冰块溶化了，冰怪也跑出来了。 

──────　│２－回到未来│　────── 

　　赶紧跑进门内，来到一间控制室，拿矿工棒子挡住通气风口的螺旋扇，来到一个通风口看见戴奇、那拉卡姆从一道太阳之门出来，并说老大王即将到来，并将我国的间谍『莫莉小姐』推进太阳之门，之後念出咒语唤出冰怪，然後又进太阳之门了看此情形马上踢开通风口的架子跳了下来，并立即拿出从ＯＴＲ２８３２中得到的一页，念出咒语制止了冰怪，但自己也因此飞起来，进入太阳之门，来到未来的世界２０３７年。 

２０３７年１月１７日－－－舒罗沙勒基地 

──────　 │１－身世之谜│ 　──────　 

　　在这里捡到枪托（右下），枪管，中段（太阳门上方）三块枪的零件及二块氮气充填器（左方）；到右方看电脑终端机，得知自己身世之谜；再到上方扫瞄器桌面上拿到蓄电池，进入扫瞄器画面，将蓄电池插进插孔，再按左下按钮，则画面出现一支枪的解说；将枪托、枪管、中段放进萤幕内，则组合成一支ＦＮＤ的枪走到最左方，用枪打碎大石，开碗柜拿到一块石头及太阳石盘将太阳石盘用在自己身上，则又回到原来的世界。 

──────　 │２－回到过去│ 　────── 

　　一回到原来世界又看见那只冰怪在活动，照样拿出那张纸念出咒文，冰怪又被镇住了跟众人说了此奇特经历後，便将众人送进太阳之门回到爱德华基地而我也进太阳之门，回到过去～布宜诺斯艾利斯图书馆救自己了回到图书馆，一切事情又重演，我马上拿出ＦＮＤ枪把哈德伦干掉，之後又传送到另一个地方了。 

１９３７年１月１８日－－－伊斯摩～那拉卡姆魔法师纪念堂 

　　来到奇怪大厅，看见左墙上有个拉铃，一拉之下，出现石碑，上面有些奇怪的图案，排法如下：（其最主要是上下二个图案要对称就可以了，那个在上，那个在下都可以）

│ 怪 │ │ 胡 │ │ 鱼神│ │ athotep └───┘ └───┘ └───┘ └───┘
　　　　　┌───┐ ┌───┐ ┌───┐ ┌───┐ │ 冰 │ │空 │ │ 水 │ │ 火 │
│ │ │ 气 │ │ │ │ │ └───┘ └───┘ └───┘ └───┘

　　结果出现一本书，书上摆了块石头，而其下方有一空位，将手上的石头摆上去，即在上方墙出现那拉卡姆、波尔斯基，而自己手中也出现一把神剑；拿剑朝二人砍砍，其二人便说了几句话後就消失了；拿走石碑上那本Ｎｅｃ－ｒｏｎｏｍｉｃｏｎ的书，门就打开了。 

────────────　│１－决斗！！叛徒西尔斯 │ ──────────── 

　　进入，坐上小船，便可看见叛徒西尔斯与其决斗，先向他问了三个问题：问有『ˇ』记号的；如问别的只会加速死亡１：苏胡人鱼神ˇ霍华帕克２：Ｎｙａｒｌａｔｈｏｔｅｐˇ约瀚Ｔ帕克Ｙｏｇｓｏｔｈｏｔｈ３：ˇ老大王戴奇Ｒ′ｌｙｅｈ之後，斩断身後的绳索，让上方灯掉下把西尔斯砸死；来到另一边洞穴，因为刚刚的爆炸而导致山洞四周都起火了。 

─────────────　│２－对决！！戴奇、那拉卡姆│　　────────── 

　　在上方墙有三个骷髅头，用剑斩破右边的骷髅头，则流出黄沙把火熄灭，走过去看见巨大怪头，但门打不开；走回三个骷髅处下方地板上有一踏板，将它踩下，门就打开了进入後来到一大片草原看见戴奇及那拉卡姆；此时那拉卡姆向我念出咒语，心觉不妙，马上使用神剑在自己身上，破了咒语，但他们仍不死心，再念咒语，但照样一来一往用神剑破了咒语最後戴奇戴上面具，而我的神剑也消失；此时将身上所剩的唯一物品Ｎｅｃｒｏｎｏｍｉｃｏｎ的书，丢向中间石头旁，则其二人也自取灭亡，掉落无尽深渊而我也被传送走了。 

──────　 │３－後记 │ 　──────　 

　　此事件虽己结束了，但不管你所选择的是过去或者未来都己不重要了；最重要的是不管做任何事都要三思而後行，免得到时为了以往的错误而奔波劳命，那就致使这五彩缤纷的一生更加疲惫不堪。
