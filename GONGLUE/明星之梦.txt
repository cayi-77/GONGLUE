明星之梦

这游戏的最大目标是要在约一年的时间，寻找 18岁以下的女孩子作为模特儿拍照，参加摄影比赛。在这短短一年期间，不单是拍照这么简单，而是要对模特儿用「心」来交往，等待她真情流露的一刻，拍下最动人的照片，就让这攻略带领你进入明星之梦……

第一部份∶四月至七月六日

　　游戏本身可以分为两大部份，其一是四月到七月六日之间，在这段时间，你从主编口中得知要参加由「 Miss Photo Jenic Contest」主办的摄影比赛，但你本身只是对拍摄风景有兴趣，所以游戏开始时摄影人物的指数偏低，而且连一个模特儿也没有，所以首要任务是先找到十八岁以下的模特儿，你可以选择主动或是被动的方式。 

　　什么是主动呢？即是你要趁休息日或星期日的时间四处寻找女孩子了，那么要到哪儿找呢？举个例子，想结识绫乃的话，先要了解其性格和出现地区。她是喜欢画画的，就应该经常会在风景区出现 ...如果你是较被动的，也不需要担心，即使你每天也只是在家中完全和外界隔绝，到某个时候你就会收到主编的电话，叫你帮他到某处做些事，而发生了与三个少女见面的事件，但常言道「种豆得豆」，照这样的方式玩下去虽然也可以爆机，却会错过很多有趣的事件。

第二部份∶七月六日至三月十四日

　　这个部份才是最重要的，是在七月六日至三月十四日之间，但在详述之前，你要先了解为何以七月六日作为界线。因为即使你在七月六日前是十分之被动，甚至完全没有出外，或是你十分努力到处追问你心仪的模特儿也好，在七月六日这天发生的事都不会改变。这天究竟发生什么事？这天是三个少女也会同时答应做你的模特儿的日子，换句话说，你之前根本不需要刻意四处找她们追问 (不怪得她们之前总是说考虑 )。在第二部份的时间，最主要是找她们来拍照 (日后比赛用的 )，但要记着，不要每个星期也约她们出外拍照，因为有很多事件是发生在其他地方，否则的话，你到游戏完结时都只得几张相片。

　　在这游戏中，取胜之道很简单。要记着最后的目标是得到摄影比赛的大奖，只要平均进行人物拍照的特训、风景拍照特训、体力的训练及打工这四项就足够了。人物的指数固然要大大提高才可以拍到高质素的人物相片，但也不要忽略风景指数，这个指数也会影响到相片出来的效果。而且在游戏最后要选相片去比赛的时候，不要只考虑到拍相片时的人物或风景指数，即使这相片的指数很高，但没有考虑到模特儿本身的性格和气质是否能配合背景的话，相片可能只是入围吧了…还有，在相片中有一项是主角对相片的评价，这个数据不能尽信，靠自己的眼光会更好。

北条 绫乃事件

日期∶ 4月 7日 (星期一 )
地点∶八甲山的掌心湖
　　今天是我接了这个计划的第一个假期，首先我的一个到的地点是丰神乐的车站，你会在这里碰到睦香之后，就到八甲山的掌心湖。突然在湖边的树下看到一个人影，原来她是你想找的其中一位模特儿 (北条绫乃 )。当日你就偷拍了她在写生的姿态，这就是你第一张拍摄的相了，恭喜、恭喜！

日期∶ 5月 4日 (星期四 )
地点∶大户美术馆 (如果之前在掌心湖见过的话，便不会发生这天的事件 )
　　到大户的美术馆看展品，在入口排队时就撞到了绫乃，她穿着一套和服，带着一把油纸伞，当想拍照的时候，她就走进美术馆。最后临走时候，我请求他做我的模特儿，但她没答我，只叫我拍一张纪念照作纪念。

日期∶ 6月 11日 (星期一 )
地点∶梦幻乐园
　　我一个人走到八甲的梦幻乐园，见到了绫乃。我就上前叫她，原来她今天到这里画画。那我作出了一个请求。就是叫她坐在花群中拍摄她画画的美姿。

日期∶ 6月 29日 (星期日 )
地点∶大户岛海岸公园
　　如果到这一天还没有认识绫乃，那你的主编又会来电，叫你到大户岛海岸公园一行。在这里你会见到绫乃在炎热的天气中累得要休息，当然你可以即时影一幅相片吧。

日期∶ 7月 5日
地点∶她就读的学校
　　到了丰神乐的北部，走着走着，就到了绫乃的学校门口，跟着就偷入学校看看，真的皇天不负有心人，终於见到绫乃在饮水机饮水，正当她把小咀凑到水龙头下的水柱，我摄影的本能又产生自然的反应了。

日期∶ 7月 12日 (星期日 )
地点∶八甲山
　　约了绫乃到八甲山上影相，不过真的不幸，绫乃不小心跌在地上，结果我们就在山上的别墅过了一晚。同日当晚，我也发了一个怪梦怪，第二天清晨 (七月二十一日)，我一醒过来，立刻就跑出屋外看看山村的清晨，那是我见到绫在树下穿了一条很美的白长衫，我就拿出相机影下来了。

日期∶ 8月 3日 (星期日 )
地点∶大户岛海岸
　　今天天很好，最适合拍一些阳光与海滩的相照片，绫乃就换了一件黑色的游衣，那就拍了这张相来。 (紧记要用高感度相片 )

日期∶ 8月 17日 (星期日 )
地点∶梦幻城前
　　我约她到梦幻公园前，我们就决定该去甚么地方，绫提议去荷兰村的迷你梦幻城。我们到了荷兰村的入口，她就换上荷兰人的服饰，我巧合地穿了荷兰人结婚的服饰。拍完相之后，我们会见到神符，如果在选择里选 (我们是在拍摄 )，那么神符就会帮你们拍下一张合照。

日期∶ 9月 21日
地点∶杂志社
　　当天你到杂志社时，突然下大雨，你本来想回公司中找伞，却会遇到绫乃，她也是因为避雨而走进来的。这么巧合，在公司中被真园睦香见到…最后可以选择送她回家。

日期∶ 10月 19日 (星期日 )
地点∶绫乃的家
　　当日我到了她家后的花园，这里非常严肃，但她说这里是人工做的，所以并不喜欢。因为她较喜欢自然感，跟着我就到了她专用作画画的室，就拍摄了她画画的姿态。

日期∶ 11月 2日 (星期日 )
地点∶大户岛海岸公园
　　我深深被海岸的环境吸引着，连绫乃站在身边也不知道，这里非常安静但不知为何没有心情去拍摄。在选择时，选「今天暂时把摄影的事丢开」，如果选了别的答案，就错过拍这张相了。后来绫乃觉得我有些说话藏在心里，她会问发生什么事的，选择「还是坦白的说吧」… 才可以把相拍了下来。

日期∶ 11月 9日 (星期日 )
地点∶八甲山梦幻乐园
　　今天天高气爽，蓝天白云，我早到了二十分钟，但是绫乃也早到了。这里有三个选择，不过一定要选看风车 (记得不要选水榭庭园和百花庭园 )但今天没有风，风车也不转了，所以就到了牧场看马。绫乃一手触摸着马儿一手扶着木栏，秀发在微风中飘扬，在这自然的美态下，我怎会错过摄影的机会呢 !?

日期∶ 12月 28日 (星期日 )
地点∶八甲山滑雪地
　　天气非常寒冷，突然有一对手在后面盖着我的眼，原来是绫乃，她还问我点解会猜得到她是谁，这里所有事物都给雪盖满，景色非常迷人。绫乃还和我打雪仗，那天我们玩得非常开心。

《 樱井爱美 》事件

日期∶游戏刚开始
地点∶丰神乐医院门口
　　游戏开始时，先到丰神乐医院。在这里会遇到一位身着制服的女孩子站在樱花树旁，而且还会趁你不为意时突然在背后推你一把…这个可爱而好动的就是「樱井爱美」。

日期∶ 6月 9日 (星期一 )
地点∶丰神公园
　　当日走到公园里，突然听到一阵阵的怪声，原来就是小美在吹 saxophone，那我就情不自禁
偷拍了她吹 saxophone的美姿。

日期∶ 6月 15日 (星期日 )
地点∶丰神乐公园
　　一个人到了公园，突然发现一个可爱活泼的小女孩，原来是小美，她主动的和我倾谈，然后与她一起走到公园里拍照。忽然落下大雨 (倒楣得很… )，我们无法再拍摄下去，只好各自回家了。糟糕 !!忘了向她说有关模特儿的事。

日期∶ 6月 22日 (星期日 )
地点∶丰神电车站
　　在电车站遇上小美，小美说想去看向日葵 (这里可以选择是否与她去看，若你不与她一同去的话，便会错失了一次拍摄的机会。 )

日期∶ 7月 13日 (星期日 )
地点∶玩具乐园
　　在之前的星期日千万不要约她影相。直到 7月 13日，到玩具乐园便会见到小美，正在聚精会神地看着一支公仔，你只要买下公仔送给她，她一定十分开心，这个真情流露的时间，又怎能不拍下来呢？

日期∶ 7月 27日 (星期日 )
地点∶大户岛海岸
　　在之前的星期日约她到大户岛海岸，当你到了大户岛海岸时，听到她在喊你的名字，但你却看不见她；原来她正在水中玩耍，她穿泳装的样子这么可爱，当然要影一幅相片吧。

日期∶ 8月 10日 (星期日 )
地点∶烟火会
　　在之前的星期日约她到烟火会。到烟火会时，她会提问一些问题，但不是太重要的。之后，当她玩烟花时便立即拍照；但有一点是十分重要的，就是一定要用高敏度底片，否则相片会黑了。

日期∶ 8月 17日 (星期日 )
地点∶八甲山的梦幻乐园
　　在之前的星期日约她到梦幻乐园，她会问你到梦幻乐园中的哪里玩，大可以放心，无论你选择甚么也不会影响结果。因为当离开时，你才会感到黄昏的美而影相。

日期∶ 11月 1日 (星期六 )或 11月 9日 (星期日 )
地点∶公园
　　你可以先到车站，有人会向你说∶「有个女孩闷闷不乐地坐在公园…」当你去到公园时会发现原来是那女孩便是小美，她还对你说心口痛，但可惜你再追问时，她却不想回答，而且连再见也没说便独自离开了…

日期∶ 11月 23日 (星期日 )
地点∶丰神乐第四高校
　　到了丰神乐第四高校时，发觉没有精神的小美一人坐在天台上，你只好把她嚷下来。下来的时候因为小美一时不小心跌下，刚好被你抱着…之后，她好像精神起来，但究竟她是否真的没事呢？还是…？

日期∶ 1月 4日 (星期日 )
地点∶滑雪场
　　在之前的星期日约她到滑雪场影相。到了滑雪场，见到她买了一块新的滑雪板，而且尝试滑雪，她却不小心跌倒，但她却没有放弃，仍继续玩下去…

日期∶ 2月 1日 (星期日 )
地点∶车站
　　在之前的星期日约她到玩具乐园影相。当天，你竟然过了约定时间才起床！你立刻赶去，但> 在车站中已经遇到了她，她叫你闭上眼睛，当你以为要被打之时竟然发觉她只是送上一吻… (这种惩罚笔者也想一试 ^_^ )

《 织原小泉 》事件

日期∶ 4月 7日
地点∶丰神乐教会
　　教堂本来就是个安严的地方。今天走到教堂，见到一个短发的人坐在教堂里，本来我以为是个男人，原来是个短发的高中女生……

日期∶五月底左右
地点∶八甲山森林区
　　只要你避免认识小泉直至五月左右，而且多数时间在家中的话，主编就会主动来电叫你到八甲山森林一行…到山上会见到一名少女正在大叫 (玩回音 !? )这时就可以立即影一幅相。但清脆的快门声却被她 (织原小泉 )发现了…

日期∶ 6月 29日 (星期日 )
地点∶电车厢里 (若开始时在教堂见过她的话，就没有这事件发生 )
　　一大清早编辑便叫我到鸣镇的影棚摄影，结果空手而回。当我乘坐电车回家的时候，在车厢里看见到一个带眼镜的女孩正在熟睡中，非常美丽。

日期∶ 7月 4日 (星期五 )
地点∶市立第一中学
　　小泉就读于丰神乐北部的一间中学，今天我在学校门口等她放学，告诉她有关模特儿的事，一边送她回家一边滔滔不绝的说着。不过最后她还没有答允我，幸好当天可以拍下一张相片。

日期∶ 8月 3日 (星期日 )
地点∶八甲山
　　在八甲山的火车站会合之后，我们就到八甲山的山林区摄影。到达山顶之后，因为时间尚早，又未到吃中午饭的时候，那就到另外一边的地方 (溪流区 )。在溪流区我们见到一支蝴蝶，小泉就上前扑蝶，结果一不小心就跌到溪水里了。

日期∶ 8月 9日 (星期六 )
地点∶市之第一高中
　　在炎炎夏日里，大家最想的东西就是水了。突听到一阵阵的水声，原来是学校里传来的。我走到水池边，见到小泉正在游泳，我的脑海只知道她还未游完这 25公尺的游泳之前，把这个画面拍下来。

日期∶ 9月 6日 (星期六 )
地点∶自己家的门口
　　正想出门的时候，看见小泉在我家附近的街道上走过，我连忙走到街上找她。到了公园，看见她闷闷不乐，原来她要在 11月 2日在学校做舞台剧，我还答应她当日到场观看。

日期∶ 9月 21日
地点∶新大户岛的美术馆
　　今天约了小泉在这摄影，小泉比我早到，从谈话之中，她露出开朗的笑容，而我也捉住了这一刻，拍下她美丽的笑容。

日期∶ 11月 2日
地点∶市立第一中学
　　还记得 9月 6日，小泉来找我，我还应承她 11月 2日会看她演出小飞侠，这天她的表现非常之好，完全不像平日的小泉。小泉出色的演出，赢得台下热烈的欢呼声。

日期∶ 11月 23日
地点∶八甲山的梦幻乐园
　　这天我约了小泉摄影，因为天气比较冷，所以就进了水榭庭园的温水泳池，正当小泉去了拿打水板教我游泳时，我便不耐烦地想着自己一个稍微练习一下，怎料竟然……，幸好最后被小泉救回。

日期∶ 12月 13日
地点∶三鸣花的巴尔逊百货公司
　　今天无无聊聊地去百货公司逛逛，进去逛了一会，觉得有点热，便到顶楼吹下风，门一打开，却看到小泉独个儿在叹气……

日期∶ 12月 28日
地点∶八甲山滑雪场
　　今天约了小泉在此摄影，由于我不懂滑雪，只好要小泉教。小泉对于滑雪很有天份，而我则滑得四脚朝天……

日期∶ 3月 8日
地点∶市立第一中学
　　小泉今天致电给我，希望我去看她的毕业典礼，不用多说，我当然答应啦。看见小泉那样子，她距离成人又近了一步，变得更加成熟了
